# Public Assets Directory

## Logo Files

Place your ScholaPulse logo files here:

1. **scholapulse-logo.png** - Full logo with text (recommended: 280x80px or 3:1 ratio)
2. **scholapulse-icon.png** - Icon only (recommended: 128x128px)

## How to Add

1. Save your logo image file
2. Rename it to `scholapulse-logo.png`
3. Place it in this `public/` folder
4. The logo component will automatically detect and use it

## File Location

```
public/
└── scholapulse-logo.png  ← Add your logo file here
```
